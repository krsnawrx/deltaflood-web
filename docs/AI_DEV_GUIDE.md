# AI Development Guide

You are assisting with development of an immersive
Three.js storytelling website called DeltaFlood.

Follow these rules strictly.

---

# Project Goal

Build an Awwwards-tier immersive site.

Do NOT simplify it into a normal website.

---

# Tech Stack

Three.js  
GSAP ScrollTrigger  
Vite  
Vanilla JS

Avoid React/Vue unless explicitly requested.

---

# Code Requirements

Always:

- modularize logic
- comment complex math
- follow performance guidelines

---

# Output Expectations

When generating code:

1. Provide full files
2. Avoid placeholders
3. Ensure compatibility with Vite
4. Use modern Three.js APIs

---

# Performance Priority

Site must run smoothly on mid-range laptops.

Avoid:

heavy shaders  
excessive geometry  
large draw calls

---

# Development Strategy

Build features incrementally:

1. Base scene
2. Earth globe
3. Satellite orbit
4. Spectral bands
5. Camera zoom
6. Flood simulation
7. AI demo
