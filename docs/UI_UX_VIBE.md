# DeltaFlood UI / UX Style Guide

The design must feel cinematic and technical.

---

# Visual Theme

Dark
Scientific
Immersive

Color palette:

Black
Deep blue
Electric cyan
Soft white text

---

# Typography

Primary font:

Inter

Usage:

Body text:
Inter

Data / metrics:
JetBrains Mono

---

# Glass UI

Overlay panels should use glassmorphism.

Example:

background: rgba(255,255,255,0.05)
backdrop-filter: blur(12px)
border-radius: 16px

---

# Motion

Animations must feel:

- smooth
- cinematic
- scroll-driven

Tools:

GSAP
ScrollTrigger

---

# UI Elements

HUD-style elements preferred.

Examples:

floating info cards  
data counters  
minimal panels

Avoid:

traditional dashboards
heavy forms

---

# Scroll Philosophy

Scrolling drives the narrative.

Examples:

scroll → satellite appears  
scroll → spectral bands split  
scroll → camera zooms into Patna

Each section must feel like a **chapter**.

---

# Audio

Sound is part of immersion.

Audio design:

First mouse movement:
space ambience fades in

Patna section:
underwater ambience

Water surface interaction:
ripple sound (throttled)

Outro:
audio fades out
